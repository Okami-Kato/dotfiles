# Prerequisites
---
## Install Alacritty

```bash
sudo pacman -S alacritty
```

## Install Git

```bash
sudo pacman -S git
```

## Install And Run Stow

```bash
sudo pacman -S stow
stow .
```

# Tmux
---
## Install Tmux

```bash
sudo pacman -S tmux
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

## Install Plugins

Press `Ctrl-Space + I` inside tmux
